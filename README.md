# FlaxOccludedRainSample
A sample of VFX rain using an occluded depth map from a secondary camera.

If you aren't sure what this is: It is a simple, low-medium cost (Depends on depth camera resolution and other factors) rain VFX that does not work under roofs and such. Here is an example video, there's no particles, no simulations, just some math.
